## **Searching and Counting Occurrences in an Array**

Write a C++ program that initializes an integer array with a fixed set of values. The program should prompt the user to enter a specific value and then perform two tasks: 
1. Determine if the value exists within the array.
2. Count and display how many times that specific value appears.

**Sample Output (Value Found)**
```text
Enter the value to search : 1
The value 1 is found 9 times in the array.
```

**Sample Output (Value Not Found)**
```text
Enter the value to search : 5
The value 5 is not found in the array.
```
