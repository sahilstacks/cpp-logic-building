**Temperature Conversion Using Functions and Menu**

Write a C++ program to perform temperature conversions using functions and a menu-driven approach.

The program should display a menu with the following options:

1. Convert Celsius to Fahrenheit
2. Convert Fahrenheit to Celsius
3. Exit

Use separate functions to perform each conversion. The program should ask the user to enter their choice and the temperature value, then display the converted result. The program should continue running until the user chooses to exit.


**Sample Output**


Temperature Conversion Menu
----------------------------------
1. Celsius to Fahrenheit
2. Fahrenheit to Celsius
3. Exit

Enter temperature in Celsius: 35
Temperature in Fahrenheit: 95
