# C++ Logic Building 🚀

A dedicated space for strengthening C++ fundamentals and logic-building through practical problem-solving. This repository contains structured solutions to common programming challenges, with a focus on clear implementations and modular code.

---

## 📂 Project Structure

Each directory contains:
- `problem.md`: A detailed description of the challenge and sample output.
- `solution.cpp`: The complete C++ implementation.

---

## 🛠️ Current Challenges

| Problem                                                       | Description |
| :-----------------------------------------------------------: | :--------------------------------------------------------------------------------------------: |
| **[Array Printing & Reversing](./array-print-and-reverse/)**  | Initialize an array and display its elements in both original and reversed order.              |
| **[Swapping Two Numbers](./swaping-two-numbers/)**            | Implement logic to swap two integers using a temporary variable and without one.               |
| **[Temperature Conversion](./temperature-conversion/)**       | A menu-driven program to convert temperatures between Celsius and Fahrenheit using functions.  |
| **[Searching and Counting Occurrences in an Array](./temperature-conversion/)**       | Creating an array, user-specified value and return its total frequency within a static array.  |
---

## 💡 Goal
The main objective of this repository is to practice low-level logic, understand data manipulation, and master C++ syntax through daily problem-solving.
